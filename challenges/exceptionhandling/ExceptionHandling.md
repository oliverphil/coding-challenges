#### Exception Handling

Divide the first number by the second and print the result to `System.out`  

Print any exceptions which occur to `System.out`

##### Example 1

Inputs:  
`x = 4`  
`y = 2`  

Output:  
`2`

##### Example 2

Inputs:  
`x = 4`  
`y = 0`  

Output:  
`java.lang.ArithmeticException: / by zero`
