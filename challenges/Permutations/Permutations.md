#### Permutations

Complete the `permutations(String s)` method to find all the permutations (anagrams) of the string `s`.  
Return a list of all the permutations as strings.

**Imports:**
* `java.util.Collections`
* `java.util.List`
* `java.util.ArrayList`


##### Example

Input:  
`s = "abc"`  

Output:  
`List<String>(abc, acb, bac, bca, cab, cba)`
