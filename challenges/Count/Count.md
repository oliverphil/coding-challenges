#### Count

Fill out the method stub to find the number of times an `item` occurs within the array `arr`

**Imports:** There are no imports included for this challenge.

##### Example

Inputs  
arr = `[1, 2, 3, 4, 1, 2, 3, 1]`  
item = `1`  

Output  
`3` (i.e. the number of times 1 occurs in the array)
